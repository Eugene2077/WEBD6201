1. bootstrap
 - vscode에서 과제폴더 열고 아래 터미널에 다음 실행(https://www.npmjs.com/package/bootstrap 여기서 아래와같은 명령어 복사해옴)
     npm install bootstrap@v5.2.3
 - 부트스트랩 설치 done!!, 이제 해당폴더에 node_module폴더/ package-lock.json 파일/ package.json파일이 설치된 것을 볼수있다.
  - 내 파일에 부트스트랩 적용하기: 적용하고 싶은 문서 <head></head>안에 아래 링크 삽입(자동설치된 node_module폴더 안에 아래 부트스트랩파일 있으니 경로 찾아서 선택하면 됨)
     <link rel="stylesheet" href="./node_modules/bootstrap/dist/css/bootstrap.min.css"

2. fontAwesome 설치
  - 아래 사이트로 가서
    https://fontawesome.com/v5/docs/web/setup/use-package-managers
 - 아래 명령어 복사
    npm install --save @fortawesome/fontawesome-free
 - vs code터미널에서 실행
  - done,이제 node_module폴더안에 @fontawesome 폴더가 설치된 것을 볼수 있다.
 - 내 파일이 fontAwesome 적용하기: 적용하고 싶은 문서 <head>안에 위와 같은방식으로 링크삽입
